Configure Latest Date Updater

How to use this extension: This extenson will query your data source for the latest date and then update a parameter to match. Create a source worksheet that has your date field set to "exact date" on detail. Use this worksheet in the configuration below. This source worksheet can be minimized to 1px by 1px if you want to hide it. Worksheet can include filters to minimize load times and/or DATETRUNC() calculations to get the right date aggregation.
